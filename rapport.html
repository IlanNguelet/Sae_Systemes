Étapes de Création et de Configuration

  1. Téléchargement et installation de VirtualBox : Rendez-vous sur le site officiel de VirtualBox, téléchargez et installez le logiciel adapté à votre système d'exploitation.
  2. Création de la machine virtuelle :
    • Ouvrez VirtualBox et cliquez sur "Nouvelle".
    • Donnez un nom à votre machine virtuelle, choisissez un dossier de sauvegarde et sélectionnez le système d'exploitation.
  3. Configuration des ressources : Allouez de la mémoire vive (RAM) et de l'espace disque en fonction des besoins du système d'exploitation.
  4. Installation du système d'exploitation : Montez une image ISO du système d'exploitation et démarrez la machine virtuelle pour commencer l'installation.
  5. Paramètres additionnels : Configurez les paramètres réseau (NAT ou Bridge) et activez les outils additionnels comme "Guest Additions" pour améliorer les performances et l'intégration.

Réponses aux Questions

Définition de la virtualisation

La virtualisation est une technologie qui permet de créer une version virtuelle d'une ressource informatique, comme un serveur, un système d'exploitation ou un réseau. Elle permet de simuler plusieurs environnements sur une seule machine physique.

Différence entre émulateur, simulateur et outil de virtualisation

  • Émulateur : Reproduit le comportement d'un matériel ou d'un système d'exploitation sur une architecture différente. Exemple : émulateurs Android.
  • Simulateur : Modélise un système pour tester son comportement dans différents scénarios sans reproduire entièrement son fonctionnement. Exemple : simulateurs réseau.
  • Outil de virtualisation : Permet d'exécuter plusieurs environnements virtuels sur un même matériel physique, en utilisant les ressources matérielles de manière partagée. Exemple : VirtualBox, VMware.

Type de virtualisation pour VirtualBox, VMware Workstation et Stratus

  • VirtualBox : Virtualisation de type 2 (virtualisation hébergée).
  • VMware Workstation : Virtualisation de type 2 (comme VirtualBox).
  • Stratus : Virtualisation de type 1 (virtualisation bare-metal).

Forme/Lieu de sauvegarde de la machine virtuelle

La machine virtuelle est sauvegardée sous forme de fichiers dans un dossier choisi lors de sa création. Les principaux fichiers incluent :

  • Fichier disque virtuel (VDI, VMDK, etc.).
  • Fichier de configuration de la VM (extension .vbox).
  • Snapshots, si activés, pour sauvegarder des états intermédiaires.

Explication et justification des choix techniques

Voici les choix techniques effectués et leurs justifications :

  • Type de système d'exploitation : Ubuntu a été choisi pour sa gratuité, sa légèreté et sa documentation abondante.
  • Allocation de ressources : 4 Go de RAM et 2 cœurs CPU pour assurer un fonctionnement fluide sans surcharger la machine hôte.
  • Type de disque dur : Disque dynamique pour optimiser l'espace utilisé sur l'hôte.
  • Mode réseau : NAT pour un accès Internet simple et sécurisé. Passage à Bridge si un accès direct au réseau est nécessaire.
